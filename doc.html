POST

/api/v1/asteroid-alerting/alert

Fetches Near-Earth Object (NEO) data for the given date range and proximity threshold, then publishes each qualifying asteroid as an individual event to the Kafka topic asteroidalerting. The notification consumer downstream processes and persists these events.

Kafka Producer asteroidalerting-producer-1
tag Kafka v4.1.1

Request Parameters

Field Type Description
start_date Required
date (ISO8601) Scanning start window. Defaults to current UTC timestamp if omitted.
end_date Optional
date (ISO8601) Scanning end window. Must be within 90 days of start date.
distance_threshold_km Required
integer Maximum proximity in kilometers for object detection alerts.
priority Optional
string (enum) Priority level for the notification.
LOW MEDIUM HIGH CRITICAL

Response Status

check_circle
200 OK

Alert triggered successfully. The scan sequence has been initiated.

error
4XX ERROR

Request failed. Check parameters or authentication credentials.

info How it works

Each asteroid matching your threshold is published as a separate message to the asteroidalerting Kafka topic. The idempotent producer (ProducerId: 5000) guarantees exactly-once delivery. The downstream notification consumer then saves each event to the database.

GET

/api/v1/asteroid-alerting/notifications

Returns persisted asteroid notification records consumed from the asteroidalerting Kafka topic. Each record represents a single NEO event that passed the alert threshold and was saved to the database by the notification consumer.

Kafka Consumer
storage MySQL

Data Schema — Notification Fields

Field Type Nullable Notes
id Primary Key
bigint NOT NULL Auto-incremented unique identifier for each notification record.
asteroid_name
varchar(255) NULLABLE Designated name of the detected near-Earth object.
close_approach_date
date NULLABLE Date of closest predicted approach to Earth.
email_sent
bit(1) NOT NULL Flag indicating whether an email alert has been dispatched for this event. 0 = No, 1 = Yes
estimated_diameter_avg_meters
double NOT NULL Average of the min and max estimated diameters of the object in meters.
miss_distance_kilometers
decimal(38,2) NULLABLE Distance in kilometers by which the object will miss Earth at closest approach.
