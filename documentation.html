Asteroid Alerting System

The Asteroid Alerting & Notification System provides programmatic access to real-time Near-Earth Object (NEO) data. This interface enables tactical teams to synchronize observation parameters with NASA data streams and trigger cascading Kafka event publishing for downstream tracking systems.

bolt

Low Latency

Direct Kafka uplink for sub-second event propagation.

radar

High Precision

Fetch astronomical data with customized distance thresholds.

history

State Sync

Maintain consistency across global monitoring nodes.

vpn_key

Authentication

All requests must include the X-Orbital-Key header. Unauthorized requests will be logged and terminated with a 401 response code.

Header: X-Orbital-Key: orb_live_7329_alpha_bravo_99
POST /api/v1/asteroid-alerting/alert

Trigger Alert Process

Manually initiates the data ingestion pipeline. The system will query the NASA NEO API for the specified date range, evaluate objects against your distance threshold, and publish matches to the configured Kafka cluster.

Request Schema

{
  "start_date": "2024-11-20",
  "end_date": "2024-11-21",
  "distance_threshold_km": 1500000,
  "priority": "CRITICAL"
}

Response Schema

{
  "status": "SUCCESS",
  "events_published": 15,
  "process_id": "uuid-0992-bx-11",
  "timestamp": "2024-11-20T14:42:01Z"
}
